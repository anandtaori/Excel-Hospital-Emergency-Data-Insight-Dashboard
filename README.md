# 🏥 Hospital Emergency Data Insights Dashboard

![Dashboard](Hospital Emergency Data Insight Dashboard.pdf)

## 📊 Project Overview
This repository showcases a **Hospital Emergency Data Insights Dashboard** that analyzes emergency department performance using visual analytics. The dashboard helps identify trends in patient visits, waiting times, satisfaction levels, and department referrals.

The visuals are created to transform raw healthcare data into **clear and actionable insights** for better decision-making.

## 🚀 Key Features
- 📌 Total number of patients analysis  
- ⏱️ Average waiting time tracking  
- ⭐ Patient satisfaction score monitoring  
- 🏥 Admission vs Non-admission status  
- 👥 Gender distribution of patients  
- 📊 Patient distribution by age group  
- 🩺 Department referral analysis  
- 📈 Daily trend analysis using area charts  

## 📂 Repository Contents
- `Hospital Emergency Data Insight Dashboard.pdf` – Contains the complete dashboard with all visual insights.

## 🛠 Tools Used
- **Excel** – Data preparation  
- **Pivot Table/Chart** – Dashboard and visual creation  

## 📈 Insights from the Dashboard
- Helps identify **busy days and patient flow patterns**
- Highlights **waiting time trends**
- Shows **department workload distribution**
- Tracks **patient satisfaction changes over time**

## 📌 Purpose of the Project
The aim of this project is to demonstrate **data visualization and analytical skills** by converting healthcare data into an easy-to-understand interactive dashboard.
